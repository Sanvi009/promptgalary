how you can Convert your boring image into this proffesional image using nano banana

This page teaches you how to generate a soft, luxurious morning selfie (pillow shot) with exact facial identity. Includes 7 lifestyle variations: after gym, beach, coffee, date night, rooftop, rainy window & artist studio.

Example of before and after result

How can you use the prompt

To create your image like this you need some stuff like ai image genaration tool ( you can use gemini, chatgpt, sora , mid journey,grok or any tool) , your image with full face visible and the promp.

First thing you need to do is upload your image and then paste the prompt , thats basic thing and you may need to setup the tool according to tool type for example on gemini you need to select create image tool and set mode to pro

you can even change the output ratio by adding ar-9:16 or ar-1:1 or any ratio you want , for more detail change check variations section bottom of the page

example result of using the exact prompt

The Prompt

A close up selfie of me with exact same facial and physical identity from the uploaded image, shirtless, with a sleepy and relaxed expression. my half open eyes look directly into the camera. The lower half of my face is buried in a luxurious white pillow, and my left bare shoulder is slightly visible. The background is a white, expensive, luxurious bedroom wall. Soft, diffused natural morning light, candid smartphone photography style, ultra realistic, 8K high resolution. Use the provided image to create an image of me. aspect ratio 9:16
