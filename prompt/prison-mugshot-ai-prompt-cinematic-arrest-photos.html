Frequently Asked Questions

  • Can I change the text on the arrest placard?
    Yes, replace (got arrested) and (NB8076) with any text you want. Keep the format similar for best results.
  • How do I make it look more realistic?
    Use a reference photo with neutral expression. Add details like 'tired eyes' or 'unshaven beard' to the prompt for more realism.
  • Will this work for female characters?
    Yes, the prompt works for any gender. Use a female reference photo and the AI will adapt the 'disheveled prisoner' description appropriately.
